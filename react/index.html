7. 定义组件（组件式什么）

7.1 组件的用法

  • React允许将代码封装成组件，然后像插入普通HTML标签一样，在网页中插入这个组件
  • 组件类的第一个字母必须大写

  • 组件类能且只能包含一个顶层标签

    7.2 组件的属性

    1.每个组件可以有自己的属性,一般用来存放组件初始后不变的数据,比如人的性别，姓名等
    2.属性一般用作组件的数据源，一般由父组件传入,比如你的名字一般是由你父母取的
    3.属性可以通过this.props中取出
    4.propTypes可以用来定义传入组件属性的名称和类型
    5.getDefaultProps函数可以用来定会引起组件的默认属性

  • 实例

7.3 this.props.children

7.4 state状态

  • 组件的状态会经常变化，只能由自己决定
  • 组件一开始有一个初始状态，然后用户互动，导致状态变化，从而触发界面重新渲染
  • getInitialState 用来定义初始状态

  • 可以给按钮绑定事件，当事件发生的时候调用对应的方法改变组件的状态

9.数据的双向绑定

  • 给组件加上ref=”xxx”后，可在父组件中通过this.refs.xxx获取该DOM元素
